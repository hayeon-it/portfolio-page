Introduce

웹 설계의 가장 기본은 마크업라고 생각합니다. 작은 것처럼 보이지만 가장 중요한 마크업을 견고히 하는, 작지만 단단한 웹퍼블리셔가 되고 싶습니다. 또한, 배움을 주저하지 않고 항상 더 나아가는, 끊임없이 발전하는 사람이 되겠습니다.

보유능력

  • HTML5 | CSS3 웹 표준 및 웹 접근성을 준수하는 시멘틱한 마크업으로 사이트를 제작합니다.
  • jQuery | Javascript 제이쿼리 문법을 활용해 동적인 효과를 구현합니다. 기본 변수, 배열, 함수 및 객체지향을 이해하고 간단한 DOM 조작이 가능합니다.
  • Ajax 비동기 방식을 활용한 페이지 구현이 가능합니다.
  • MySQL | Php 데이터베이스를 활용한 서버페이지 제작이 가능합니다.
